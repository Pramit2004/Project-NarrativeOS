# 📚 TxtReader Legend - The Ultimate Text File Reader

Transform any text file into a premium reading experience with AI-powered analysis, beautiful typography, and smart navigation.

## ✨ Features

### 📖 Reading Experience
- **Beautiful Typography** - 10+ premium fonts with perfect spacing
- **6 Handcrafted Themes** - Cream, Noir, Dusk, Forest, Slate, Ocean, Sepia
- **Responsive Design** - Perfect on phones, tablets, and desktops
- **Focus Mode** - Distraction-free reading
- **Typewriter Mode** - Center current line focus

### 🤖 AI-Powered (Gemini 2.5 Flash)
- **Deep Document Analysis** - Extract themes, characters, writing style
- **Chat with Your Document** - Ask questions about the content
- **Smart Summaries** - Get executive summaries and detailed synopses
- **Quiz Generation** - Test your understanding
- **Key Quotes & Themes** - AI identifies important passages

### 🎨 Interaction
- **6 Highlight Colors** - Color-code important passages
- **Bookmarks** - Save your place anywhere
- **Notes** - Attach notes to any paragraph
- **Search with Highlighting** - Find text instantly
- **Selection Popup** - Quick AI actions on selected text

### 📊 Analytics
- **Word/Character Counts** - Detailed statistics
- **Reading Time** - Accurate estimates
- **Lexical Diversity** - Vocabulary richness
- **Top Words** - Frequency analysis
- **Progress Tracking** - Visual progress bar

### 📤 Export Options
- **Formatted HTML** - Beautiful, styled document
- **Markdown** - Clean markdown export
- **AI Analysis Report** - Comprehensive text report
